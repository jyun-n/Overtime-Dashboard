# 연장 관리 대시보드 — 사용자 매뉴얼

본 문서는 **일반 사용자(USER 권한)** 를 대상으로 합니다. 관리자(ADMIN)는 [관리자_매뉴얼.md](./관리자_매뉴얼.md)를 참고하세요.

---

## 1. 접속 / 로그인

1. 사내망 PC에서 브라우저로 대시보드 주소에 접속합니다.
2. 관리자에게 발급받은 **아이디 / 비밀번호** 를 입력합니다.
3. **아이디 저장** 체크 시 다음 접속부터 아이디가 자동 입력됩니다.
4. 로그인에 성공하면 **연장 관리 대시보드** 화면으로 이동합니다.

### 로그인 오류 안내

| 메시지 | 의미 / 조치 |
| --- | --- |
| 아이디 또는 비밀번호가 올바르지 않습니다. | 입력 정보를 다시 확인합니다. |
| 로그인 시도가 너무 많습니다. 15분 후 다시 시도해주세요. | 짧은 시간 내 실패가 누적된 상태입니다. 15분 후 재시도하거나 관리자에게 문의하세요. |
| 요청이 일시적으로 많아졌습니다. 잠시 후 다시 시도해주세요. | 일시적인 과부하입니다. 잠시 후 다시 시도합니다. |
| 로그인 중 오류가 발생했습니다. | 네트워크/서버 문제일 수 있습니다. 관리자에게 문의하세요. |

### 로그아웃
화면 우측 상단의 **로그아웃** 버튼을 누르면 즉시 로그인 페이지로 이동합니다.

---

## 2. 대시보드 기본 화면

상단 영역에 다음 정보가 표시됩니다.

- **인사정보 파일 / 연장근무 데이터** : 마지막으로 업로드된 시점
- **이름 · 부서 · IP** : 현재 로그인한 본인 정보
- 우측 상단의 탭으로 **종합 / 상세** 보기를 전환합니다.

### 데이터 기준
- 모든 분석은 관리자가 업로드한 **인사정보** + **연장근무 엑셀** 데이터에 근거합니다.
- 데이터가 비어 있거나 최신화되지 않은 경우 관리자에게 문의하세요.

---

## 3. 종합(Overview) 탭

### 3-1. 상단 필터

| 항목 | 설명 |
| --- | --- |
| **직군 필터** | 전체 직군 / 간호직 / 교수직 / 기능직 / 사무직 / 시설직 / 약제직 / 영양직 / 의료기술직 / 전공의 / 행정직 중 선택 |
| **기간 설정** | 시작 월 ~ 종료 월을 직접 선택. 단일 월 선택도 가능 |

> 초기 진입 시 **직전 달** 단일월이 기본으로 설정되며, 이때 KPI 카드의 **전월 대비** 증감률이 함께 표시됩니다. 기간을 변경하면 합산 값만 표시됩니다.

### 3-2. KPI 카드 (5개)

| 카드 | 의미 |
| --- | --- |
| 총 연장시간 | 자동 연장 + 초과 연장 합산 시간 |
| 총 연장수당 | 자동 + 초과 수당의 합 (단위: 천만원) |
| 초과연장시간 | 한도를 초과한 부분의 시간 |
| 초과연장수당 | 초과 부분의 수당 (단위: 천만원) |
| 인당 평균 | **월평균 1인 연장시간** — 선택 기간의 총 연장시간을 (연장근무 기록이 있는 인원 × 개월수)로 나눈 값. 기간을 늘려도 단위가 유지됩니다. |

전월 대비 증감은 빨간색(↑, 증가) / 초록색(↓, 감소) / 회색(동일) 으로 표시됩니다.

### 3-3. 좌측 차트 패널

- **연장시간 차트** — 자동 / 초과 영역을 누적한 상태로, 총연장(빨간 선)이 함께 표시됩니다.
- **연장수당 차트** — 자동 / 초과 바를 누적, 총합(빨간 선)을 표시. 단위는 천만원.
- 차트 위에 마우스를 올리면 월별 정확한 값을 툴팁으로 확인할 수 있습니다.

> **기간이 단일 월** 일 경우, 추세 확인을 위해 차트만 직전 4개월로 자동 확장됩니다(KPI/표는 선택한 월 기준).

### 3-4. 부서 TOP 20

선택한 기간의 부서별 누적치를 자동연장 / 초과연장으로 나누어 상위 20개 부서를 막대로 표시합니다.

- 각 부서 행을 **클릭** 하면 해당 부서의 월별 추이 모달이 열립니다.
- 모달 내에서 기간을 변경하여 부서별 장기 추이를 확인할 수 있습니다(기본 직전 4개월).

### 3-5. 개인 TOP 15

선택한 기간 동안 **초과(시간)** 가 가장 큰 상위 15명을 표시합니다.

- **부서 필터 버튼** 으로 표시 대상 부서를 다중 선택 가능. 검색창에서 부서명을 입력해 빠르게 찾을 수 있습니다.
- 차트의 막대(초과 시간) 또는 이름을 **클릭** 하면 해당 인원의 월별 초과 / 금액 추이 모달이 열립니다.

### 3-6. 종합 리포트 다운로드

우측 상단의 **다운로드 버튼(종합 탭 활성 시)** 을 누르면, 현재 화면에 표시 중인 KPI · 차트 · TOP 부서 · TOP 개인 정보를 종합한 엑셀 리포트가 저장됩니다.

## 4. 상세(Detail) 탭

부서·직군별 누적 데이터를 더 세밀하게 분석할 수 있는 화면입니다.

- 직군 / 부서 / 연장 유형(자동연장, 초과연장, 총연장) 등 다중 필터 제공
- 월별 추이를 막대 + 금액 라인으로 동시 시각화
- 우측 상단의 **다운로드 버튼(상세 탭 활성 시)** 으로 현재 표시된 상세 데이터를 엑셀로 저장할 수 있습니다.

> 탭을 전환해도 이전 탭의 상태(필터·기간)는 그대로 보존됩니다.

## 5. FAQ / 자주 묻는 질문

**Q. 데이터가 비어 있어요.**
A. 관리자가 인사정보 또는 연장근무 파일을 아직 업로드하지 않은 경우입니다. 상단의 "인사정보 파일 / 연장근무 데이터 — 업데이트 없음" 표시를 확인하고 관리자에게 문의하세요.

**Q. 비밀번호를 잊었습니다.**
A. 본인 PC에서 직접 변경할 수 없습니다. **관리자에게 비밀번호 재설정** 을 요청하세요.

**Q. 전월 대비 % 가 안 보입니다.**
A. 전월 대비는 **기본값(직전 단일월)** 일 때만 표시됩니다. 기간을 변경하면 합산만 표시됩니다.

**Q. 차트의 단위가 헷갈립니다.**
A. 시간 차트 = **시간**, 수당 차트 = **천만원**, 개인 TOP 차트의 금액 라인 = **천원** 입니다. 각 패널 우측에 단위가 함께 표시됩니다.

**Q. 사내망 외부에서 접속할 수 있나요?**
A. 시스템은 사내망(0.0.0.0 바인딩) 전용입니다. 외부에서는 접속할 수 없습니다.
